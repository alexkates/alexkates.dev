# alexkates.dev

The source for [alexkates.dev](https://alexkates.dev), including the complete blog archive and its images.

## Development

```console
bun install
bun dev
```

Blog posts live in `content/blog` as Markdown with validated frontmatter. Article images live in `public/blog/<slug>`.

## Verification

```console
bun lint
bun run build
```
